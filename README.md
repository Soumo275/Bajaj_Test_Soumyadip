# Bajaj_Test_Soumyadip

Bajaj Test submission

This Spring Boot application calls a webhook API, retrieves a JWT token, and submits a SQL query.

## Prerequisites

- Java 18 or higher
- Maven

## Build & Run

1. **Build the project**

````bash
mvn clean package

2. **Run the JAR**

```bash
java -jar target/soumo275-0.0.1-SNAPSHOT.jar


The application will:

Call the generateWebhook API.

Print the webhook URL and JWT token.

Submit the final SQL query to the webhook.

Print the webhook response.
````
